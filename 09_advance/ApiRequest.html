XMLHttpRequest ReadyState Explanation

This example demonstrates how XMLHttpRequest works and how readyState changes during an API request.

ReadyState Values

Value State Description
0 UNSENT Client has been created. open() not called yet.
1 OPENED open() has been called.
2 HEADERS_RECEIVED send() has been called, headers and status are available.
3 LOADING Downloading; responseText holds partial data.
4 DONE The operation is complete.

What to do

  • Open the browser console
  • Press F12
  • Go to the Console tab
  • Watch readyState values being logged
